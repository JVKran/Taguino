const unsigned int HCSR04::getDistance ( const bool  resultInInches = false)

Get Distance.

This function returns the distance in centimeters. The triggerpin is first made low to be sure a clean signal is made when it is made high 5us later. The datasheet specifies the signal has to be high for at least 10us. But before a signal can be measured, we first have to wait for the bin to become high. Then the timing is started and stopped when the pin becomes low. The speed of sound is equal to 1 cm per 29us hence the distance can be calculated as implemented.
