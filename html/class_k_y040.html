Constuctor.

This constructor has two mandatory parameters; the CLK and DT pin. Without these pins it is impossible to determine the position of the rotary encoder. The SW doesn't have to be connected; it then always returns 0.

int KY040::getPos ( )

Get Position.

This function returns the current position of the encoder. For the value to be recent, the update() function has got to be called often enough. Positions can be both, negative (counter clockwise) and positive (clockwise).

◆ update()

void KY040::update ( )

Update readings.

This function determines the position of the encoder and checks wether the button is pressed. Since Rotary Encoders are completely time-based, this function has got to be called frequently enough for the values to be updated correctly (~10ms / ~80ms). The Rotary Encoder outputs a grey-code and this function tries to interpret that outputted code. It's very basic though.
