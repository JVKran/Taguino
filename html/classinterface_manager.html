Encoder Turned.

This function has one mandatory parameter; the current position. One rotaryEncoder step creates two pulses; hence we only want to know when the position is even. Then, we want to divide that by two to simulate a 0, 1, 2, 3 steppattern when in reality it would be 0, 2, 4, 6, etc.

Implements encoderListener.

Interface Managing Task.

This task has a main() which is responsible for handling user input. For example, when the position % 2 == 1 the weapon is being selected. Then, when the encoder is turned, another weapon is chosen. A function is then called to change the weapon.
